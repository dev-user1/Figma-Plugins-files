## English
This repository contains the files required to run the **Colorized** plugin. 
The plugin is designed to enhance color management in various applications. 
Make sure to follow the installation instructions to get started with **Colorized**.

For more information, visit our Figma page: [Fentago Design](https://www.figma.com/@fentago)
